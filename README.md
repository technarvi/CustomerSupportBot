# CustomerSupportBot
This is a chatbot through mobile app


Tools:
*Android studio
*Chatfuel
*Facebook Messenger
*Firebase(for login and validation)

In this project i have used a mobile app through which the end-user can communicate to the company regarding any customer support service or issuse.It can be a communication with the Customer care or the ChatBot.I have used firebase for validation.Bot is under the stage of Training
